# A Systematic Comparative Study of Supervised and Deep Learning Models for Bitcoin Price Prediction Using Historical Market Data

**Student ID:** U2940362

## 📌 Overview

This project presents a systematic comparative analysis of traditional supervised machine learning models and deep learning approaches for predicting Bitcoin price movements using historical market data. The goal is to evaluate model performance, identify strengths and limitations, and determine which techniques are most effective for financial time-series forecasting.
